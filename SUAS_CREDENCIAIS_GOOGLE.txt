# 🔐 Suas Credenciais Google OAuth

## ✅ Client ID
(obtenha em Google Cloud Console > APIs & Services > Credentials)

## ✅ Client Secret
(obtenha na mesma página; guarde em local seguro e não coloque no repositório)

---

## 📋 O que fazer agora:

### 1. No Google Cloud Console:
   - Volte para a tela onde viu o Client ID
   - Copie o **Client Secret** (aparece apenas uma vez!)

### 2. No Render.com:
   - Acesse: https://dashboard.render.com
   - Seu serviço > "Environment"
   - Adicione:
     - Key: GOOGLE_CLIENT_ID
     - Value: (o Client ID do seu projeto Google)
   
   - Adicione:
     - Key: GOOGLE_CLIENT_SECRET
     - Value: [cole o Client Secret aqui]

### 3. Verificar URLs de Callback:
   No Google Cloud Console, nas credenciais OAuth, certifique-se de ter estas URLs:
   
   ✅ https://conectaking-api.onrender.com/api/oauth/agenda/google/owner/callback
   ✅ https://conectaking-api.onrender.com/api/oauth/agenda/google/client/callback

### 4. Fazer Deploy:
   - No Render, clique em "Manual Deploy" > "Deploy latest commit"

### 5. Testar:
   - Dashboard > Agenda Inteligente > Conectar Google Calendar
   - Deve funcionar! ✅

---

## 🔒 IMPORTANTE:
- NUNCA compartilhe o Client Secret publicamente
- Mantenha essas credenciais seguras
- O Client Secret só aparece uma vez no Google Cloud Console
