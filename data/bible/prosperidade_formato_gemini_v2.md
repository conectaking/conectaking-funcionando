# Formato Gemini v2 — mapeamento automático

| No texto colado | Campo no admin |
|-----------------|----------------|
| `ATIVAÇÃO 01: …` | Título |
| Citação `"…" — KING` (logo abaixo) | Decreto de entrada |
| `1. O FUNDAMENTO SAGRADO:` | Fundamento sagrado |
| `2. EXTRAÇÃO DE PROSPERIDADE:` | Extração de prosperidade |
| `FRASES DE IMPACTO DO KING:` | Frases de impacto (IE chave) |
| `3. NA ESTRADA COM O KING:` | Na estrada com o KING |
| `DIRETRIZ DE ILUSTRAÇÃO` | Diretriz de ilustração |
| `4. REPROGRAMAÇÃO MENTAL DE IMPACTO` | Mentalidade travada / Nova mentalidade / Exercício |
| `5. O TREINO DO REI` | Treino negócios + Treino altar |
| `7. ATIVAÇÃO COMPLEMENTAR` | Anexado ao Treino altar |
| `6. SENTENÇA DE ATIVAÇÃO DIÁRIA` | Sentença de ativação |
| `PRÓXIMO EPISÓDIO:` | Próximo episódio |

Colar o texto completo → **Dividir seções** ou **Salvar ativação**.
