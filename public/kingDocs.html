King Docs — cofre privado: estes dados não aparecem no teu cartão público. Só saem daqui quando crias um link temporário ou exportas / copias tu mesmo.
Como funciona o cofre? (ajuda rápida)
  • Dados — preenche e guarda; podes importar do teu cartão Conecta King ou exportar PDF como backup.
  • Documentos — envia PDFs ou imagens; ficam guardados até apagares.
  • Criar link — escolhe o que entra no link; o destinatário só vê isso. Define prazo, senha ou limite de vistas.
  • Os meus links — copia URLs ou revoga quando não precisares. Nada disto envia e-mail automaticamente.

  • Depois de gerado, podes revogar o link na aba «Os meus links».
